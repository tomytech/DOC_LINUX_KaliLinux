Téléchargement de Cloudflare

1. Passez en mode Administrateur :

( user@kali )-[ ~ ]
sudo su

3. Nettoyez la console :

( root@kali )-[ /home/user ]
clear

4. Téléchargez Cloudflare pour Linux à l'aide de la commande wget :

( root@kali )-[ /home/user ]
wget https://github.com/cloudflare/cloudflared/releases/download/2023.1.0/cloudflared-linux-amd64

Si vous voulez une autre version : https://github.com/cloudflare/cloudflared/releases
(Sélectionnez la version -> clique droit -> copier l'adresse du lien)

5. Nettoyez la console :

( root@kali )-[ /home/user ]
clear

6. Vérifiez que le fichier cloudflared-linux-amd64 a bien été téléchargé :

( root@kali )-[ /home/user ]
ls -la

7. Attribuez les droits d'exécution à ce fichier :

( root@kali )-[ /home/user ]
chmod +x cloudflared-linux-amd64

8. Vérifiez que l'attribution a été bien été faite :

( root@kali )-[ /home/user ]
ls -la

9. Nettoyez la console :

( root@kali )-[ /home/user ]
clear

Lancer Cloudflare

Demander à Cloudflare de générer une URL qui redirige vers localhost:80 :

( root@kali )-[ /home/user ]
./cloudflared-linux-amd64 tunnel --url localhost:80

./ : pour exécuter un script

tunnel : pour créer un tunnel externe

--url : pour générer une URL

localhost:80 : port du serveur local écouté 80

Sortir de l'exécution avec la commande ctrl c

Autorisez les groupes d'utilisateurs root et daymon à faire un ping depuis le monde extérieur :

( root@kali )-[ /home/user ]
sysctl net.ipv4.ping_group_range='0 1'

Lancer Apache

Installation d'Apache :

( root@kali )-[ /home/user ]
apt install -y apache2

Démarrer le serveur web d'Apache :

( root@kali )-[ /home/user ]
service apache2 start

Si vous allez sur un navigateur internet et que vous saisissez dans la barre d'url l'adresse IP de la machine virtuelleKaliLinux obtenu à l'aide de la commande ip a -> eth0 -> inet (sans le /24), vous devriez arriver sur la page d'accueil d'Apache2.

Rendre la page d'accueil d'Apache accessible depuis le monde extérieur

1. Ouvrez le fichier index.html de la page d'accueil d'Apache :

( root@kali )-[ /home/user ]
nano /var/www/html/index.html

2. Dans le fichier, au niveau de Apache2 Debian Default Page, rajoutez IRIS avant Debian

Enregistrez la modification avecctrl x->O (la lettre)->Entrée

3. Actualisez la page web d'Apache sur un navigateur internet.

4. Redemandez à Cloudflare de générer une URL (un tunnel externe) qui redirige vers localhost:80 :

( root@kali )-[ /home/user ]
./cloudflared-linux-amd64 tunnel --url localhost:80

Une URL va être générer dans l'exécution de la commande et sera encadré.

5. Sélectionnez l'URL généré puis faite un clique droit et cliquez sur Copier l'adresse du lien.

6. Collez l'URL copié dans la barre d'url d'un navigateur internet.

Vous devriez arriver sur la page d'accueil d'Apache (grâce ducoup à une URL générée).

Ouvrez un nouveau Terminal et identifiez-vous de nouveau en tant qu'administrateur -> sudo su -> root

L'outils setoolkit

setoolkit est un utilitaire qui permet de faire pas mal de choses, comme du Social Engineering à travers la mise en place du fichier.

Lancement de l'outils :

( root@kali )-[ /home/user ]
setoolkit
