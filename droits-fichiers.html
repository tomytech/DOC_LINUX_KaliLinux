Attribution de(s) droit(s)

Syntaxe

chmod [N1][N2][N3] nom_fichier.txt

[N1]-> Droits attribués au propriétaire du fichier

[N2]-> Droits attribués au groupe propriétaire du fichier

[N3]-> Droits attribués aux autres propriétaires du fichier

Droit Value Octale
r Lecture 4
w Écriture 2
x Exécutable 1
Attribution Lecture Écriture Exécutable S
Propriétaire
Groupe propriétaire
Autres propriétaires 
chmod [N1][N2][N3] nom_fichier.txt

Attribuer tous les droitsrwxà un fichier :

( root@kali )-[ /home/user/ ]
chmod 777 nom_fichier.txt

Attribuer seulement le droit de lecturerd'un fichier :

( root@kali )-[ /home/user/ ]
chmod 444 nom_fichier.txt

Attribuer seulement le droit d'écriturewd'un fichier :

( root@kali )-[ /home/user/ ]
chmod 222 nom_fichier.txt

Attribuer seulement le droit d'exécutionxd'un fichier :

( root@kali )-[ /home/user/ ]
chmod 111 nom_fichier.txt

Attribuer le droit de lecture en plusrà un fichier :

( root@kali )-[ /home/user/ ]
chmod +r nom_fichier.txt

Attribuer le droit d'écriture en pluswà un fichier :

( root@kali )-[ /home/user/ ]
chmod +w nom_fichier.txt

Attribuer le droit d'exécution en pluswà un fichier :

( root@kali )-[ /home/user/ ]
chmod +x nom_fichier.txt

Enlever tous les droits d'un fichier :

( root@kali )-[ /home/user/ ]
chmod 000 nom_fichier.txt
