Téléchargement de Villain

1. Allez dans le dossier Téléchargements :

( user@kali )-[ ~ ]
cd Téléchargements/

2. Passez en mode Administrateur :

( user@kali )-[ ~/Téléchargements ]
sudo su

3. Nettoyez la console :

( root@kali )-[ /home/user/Téléchargements ]
clear

4. Clonez le lient githttps://github.com/amirouchef/Villain.git :

( root@kali )-[ /home/user/Téléchargements ]
git clone https://github.com/amirouchef/Villain.git

5. Vérifiez que le dossier Villain a bien été téléchargé :

( root@kali )-[ /home/user/Téléchargements ]
ls -la

6. Accédez au dossier Villain téléchargé :

( root@kali )-[ /home/user/Téléchargements ]
cd Villain/

Dossier Villain

Voir le contenu du dossier :

( root@kali )-[ /home/user/Téléchargements/Villain ]
ls -la

Voir le contenu d'un fichier :

( root@kali )-[ /home/user/Téléchargements/Villain ]
